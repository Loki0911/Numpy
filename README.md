# NumPy for Data Science

A collection of Jupyter notebooks covering NumPy — Python's core library for numerical computing — with a focus on practical data science usage.

## Notebooks

| # | Notebook | Description |
|---|----------|-------------|
| 1 | `numpy_fundamentals.ipynb` | Core concepts: arrays, dtypes, indexing, slicing, and basic operations |
| 2 | `more of numpy.ipynb` | Intermediate topics: broadcasting, reshaping, and mathematical functions |
| 3 | `numpy_tricks.ipynb` | Handy tips and patterns for writing efficient, idiomatic NumPy code |

## Getting Started

```bash
git clone https://github.com/Loki0911/Numpy.git
cd Numpy
pip install numpy jupyter
jupyter notebook
```

## Requirements

- Python 3.x
- NumPy
- Jupyter Notebook
